En esta sección trabajaremos en React creando una aplicación capaz de consumir nuestro WebSocket server, puntualmente aprenderemos sobre:

- Comprender cómo funciona una aplicación en tiempo real de extremo a extremo
- Integrar WebSockets nativos en React usando Context y hooks personalizados
- Manejar conexiones, reconexión y estados de red.
- Enviar, recibir y tipar mensajes en tiempo real.
- Actualizar la interfaz de usuario en vivo a partir de eventos del servidor
- Probar y validar el comportamiento en múltiples navegadores
